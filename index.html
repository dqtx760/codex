认识 Codex

什么是 Codex

Codex 是 OpenAI 推出的AI编程助手，能够理解你的整个代码库，独立完成复杂的编程任务。

核心能力：

  • 多代理并行：同时处理多个任务，效率倍增
  • 代码库理解：读懂整个项目，而非单个文件
  • 任务执行：从需求到交付的完整闭环

与传统工具对比

特性传统工具Codex
工作方式单次交互多步骤任务
上下文当前文件整个代码库
输出代码片段完整PR
02

安装准备

账号要求

需要 ChatGPT Plus 或 Pro 账号才能使用完整功能。

安装方式

网页版：访问 chatgpt.com/codex，无需安装

桌面应用：从 OpenAI 官网下载安装包

命令行：npm install -g @openai/codex

03

界面导览

三大区域

  • 侧边栏：管理所有任务线程
  • 主工作区：查看任务进度和结果
  • 审查面板：检查代码变更
04

核心流程

四步工作法

  1. 规划：明确目标、分析现状、制定方案
  2. 委派：提供清晰的任务描述
  3. 验证：检查代码变更和测试结果
  4. 审查：确认无误后合并

任务描述模板

prompt
目标：[要做什么] 上下文：[相关文件和背景] 约束：[技术限制] 验证：[如何确认完成]

高级技巧

AGENTS.md 项目规则

在项目根目录创建 AGENTS.md 文件，定义代码规范、测试命令等，让 Codex 始终遵循你的规则。

技能复用

将常用任务打包为技能，一键复用，保持一致性。

CodexPlusPlus

增强工具

CodexPlusPlus（17.8k Stars）是 Codex App 的增强工具。

核心功能：

  • 中转注入：接入国产模型API
  • 插件解锁：解锁被限制的功能
  • 会话管理：删除、导出、移动

接入国产模型

通过中转注入，可以使用国产模型替代官方API，降低成本。

配置步骤
1. 下载安装 CodexPlusPlus 2. 打开管理工具 → 中转注入 3. 添加 Base URL 和 API Key 4. 应用配置并启动
