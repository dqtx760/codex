01

认识 Codex

Codex 是 OpenAI 推出的AI编程助手，能够理解整个代码库，独立完成复杂任务。

多代理并行

同时处理多个任务，效率倍增。一个代理重构代码，另一个修复bug。

代码库理解

读懂整个项目结构，理解依赖关系、代码风格和业务逻辑。

任务执行

从需求到交付的完整闭环。自动修改代码、运行测试、生成PR。

02

安装准备

账号要求

需要 ChatGPT Plus 或 Pro 账号。推荐使用 Pro 获得完整功能。

网页版

访问 chatgpt.com/codex，无需安装，立即开始。

bash
npm install -g @openai/codex
03

界面导览

Codex 界面分为三大区域，各有分工。

侧边栏

管理所有任务线程。创建、切换、查看状态。

主工作区

查看任务进度、代码变更、测试结果。

审查面板

检查代码变更，支持并排对比。

04

核心流程

掌握四步工作法，让 Codex 成为得力助手。

1. 规划

明确目标、分析现状、制定方案。

2. 委派

提供清晰的任务描述。

3. 验证

检查代码变更和测试结果。

4. 审查

确认无误后合并。

prompt
目标：[要做什么] 上下文：[相关文件和背景] 约束：[技术限制] 验证：[如何确认完成]
05

高级技巧

AGENTS.md

在项目根目录创建规则文件，定义代码规范、测试命令等。

技能复用

将常用任务打包为技能，一键复用，保持一致性。

06

问题排查

启动失败

删除配置文件夹（~/.codex），重新登录。

登录循环

清除浏览器缓存，检查账号权限。

网络错误

检查代理设置，确认网络连接。

12

Windows 排障

启动报错 0xC0000022

沙盒权限问题。修改配置切换模式。

文件系统错误

应用商店组件异常。重置缓存修复。

Reconnecting

代理配置问题。设置环境变量。

需要专业帮助？

复杂问题可能需要一对一诊断。扫码添加微信获取远程协助。

13

CodexPlusPlus

17.8k Stars 的增强工具，解锁更多能力。

中转注入

接入国产模型API，降低成本。

插件解锁

解锁原生被限制的功能。

会话管理

安装步骤
1. 从 GitHub Releases 下载安装包 2. 打开 Codex++ 管理工具 3. 配置中转注入（Base URL + API Key） 4. 启动 Codex++ 即可使用
